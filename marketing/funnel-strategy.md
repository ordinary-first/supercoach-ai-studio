# Secret Coach AI - 가치 사다리 & 퍼널 전략 문서

> 작성 기준: 이상한마케팅 고전환 프레임워크
> 최종 업데이트: 2026-02-19

---

## 1. 현재 퍼널 진단

### AS-IS (현재 상태)
```
[직접 URL 접속] → [랜딩페이지] → [Google 로그인] → [온보딩/플랜 선택] → [결제]
```

**문제점:**
- 유입 채널이 "직접 접속"뿐 — 트래픽 소스 0개
- 리드 자석(Lead Magnet) 부재 — 잠재 고객 정보 수집 불가
- "인지 → 관심 → 결제" 사이의 신뢰 구축 단계가 없음
- 가격 비교/경쟁사 대비 차별점을 인지시킬 콘텐츠 없음

### TO-BE (목표 상태)
```
[콘텐츠 유입] → [리드 자석] → [무료 체험] → [Essential 전환] → [Visionary 업그레이드] → [충성 고객/추천]
      ↑              ↑              ↑              ↑                     ↑
   블로그/SNS    PDF/뉴스레터    3일 체험     $9.99/월           $19.99/월
```

## 2. 가치 사다리 (Value Ladder) 상세 설계

### 2-1. 1단계: 유입 (Traffic) — 무료 콘텐츠

**목적:** 타겟 고객이 검색하거나 SNS에서 발견할 수 있는 콘텐츠로 브랜드 인지도 형성

| 채널 | 콘텐츠 유형 | 주요 키워드/주제 | 게시 빈도 |
|------|------------|-----------------|----------|
| 블로그 (네이버/티스토리) | 장문 칼럼 (2,000자+) | "목표 설정 방법", "작심삼일 극복", "시각화 기법" | 주 2~3회 |
| 인스타그램 릴스 | 15~30초 숏폼 | "작심삼일 극복 3초 팁", "뇌과학 목표 설정" | 주 3~5회 |
| 유튜브 숏츠 | 30~60초 숏폼 | "AI로 꿈 시각화하기", "성공하는 사람들의 목표 관리법" | 주 2~3회 |
| 카카오 뷰/브런치 | 에세이형 칼럼 | "인생 목표를 마인드맵으로 정리하는 법" | 주 1회 |

**콘텐츠 설계 원칙 (이상한마케팅 기법 적용):**
- **후크 3초 법칙**: 첫 문장/첫 화면에서 고통(Pain Point)을 정확히 찌름
- **반박 제거**: 콘텐츠 중간에 "이게 진짜 돼?" 같은 의심을 미리 해소
- **CTA 삽입**: 모든 콘텐츠 말미에 무료 PDF 또는 앱 체험 유도

**핵심 콘텐츠 주제 10개:**
1. "작심삼일이 반복되는 뇌과학적 이유 (그리고 해결법)"
2. "성공한 사람들이 목표를 '보는' 방법 — 시각화의 과학"
3. "AI가 당신의 꿈을 이미지로 만들어주면 뇌에서 일어나는 일"
4. "마인드맵으로 인생 목표 정리하는 5단계 가이드"
5. "왜 메모장 목표 관리는 100% 실패하는가"
6. "하루 3분, 뇌의 RAS를 해킹하는 시각화 루틴"
7. "'올해는 다를 거야' — 이 다짐이 매년 실패하는 구조적 이유"
8. "목표 달성률 300% 높이는 AI 코칭의 비밀"
9. "1인 기업가가 반드시 써야 할 목표 관리 시스템"
10. "당신의 목표가 구체적이지 않으면 뇌는 실행을 거부한다"

### 2-2. 2단계: 리드 (Lead) — 리드 자석

**목적:** 잠재 고객의 이메일/카카오톡 채널 구독을 확보하여 재접촉 가능한 관계 형성

| 리드 자석 | 형태 | 타겟 | 수집 정보 |
|----------|------|------|----------|
| "뇌과학 기반 목표 달성 가이드" PDF | 15~20페이지 전자책 | 자기계발/목표관리 관심자 | 이메일 |
| "7일 목표 시각화 챌린지" 이메일 시리즈 | 7일간 자동 이메일 | 시각화/자기계발 관심자 | 이메일 |
| 카카오톡 채널 "매일 목표 알림" | 매일 1회 동기부여 메시지 | 한국어 사용자 전체 | 카카오톡 채널 추가 |

**PDF 리드 자석 구조 (후크→스토리→오퍼):**

```
목차 구성:
1. [후크] 왜 당신의 목표는 항상 3일 만에 사라지는가
2. [문제 심화] 의지력의 과학적 한계 — 뇌는 변화를 싫어한다
3. [해결책 제시] 시각화가 뇌의 RAS를 재프로그래밍하는 원리
4. [실전 가이드] 마인드맵으로 인생 비전을 설계하는 5단계
5. [심화] AI 시각화로 뇌에 성공 이미지를 각인하는 방법
6. [사회적 증거] 실제 사용자들의 변화 사례
7. [CTA] Secret Coach로 지금 바로 시작하기 (3일 무료)
```

**리드 자석 배포 채널:**
- 블로그 글 말미: "이 글의 풀 버전을 PDF로 받아보세요"
- 인스타그램 릴스 프로필 링크
- 랜딩페이지 중간 삽입 (향후 구현 시)
- 유튜브 영상 설명란

### 2-3. 3단계: 진입 (Entry) — 무료 체험

**목적:** 첫 결제 경험 없이 제품 가치를 체험하게 하여 심리적 장벽 제거

**현재:** Explorer 플랜 (3일 무료 체험, 카드 등록 없음)

**최적화 전략:**
- 온보딩 과정에서 "첫 목표 + 첫 시각화"를 강제 완성시킴 (가치 체험 최소화)
- 체험 2일차에 인앱 알림: "내일이면 체험이 종료됩니다. 지금까지 세운 목표 X개를 잃지 마세요" (손실 회피)
- 체험 3일차 만료 시: "당신이 세운 목표와 시각화를 영구 보존하려면 플랜을 업그레이드하세요"
- 이메일 시퀀스 (체험 중 3통):
  - Day 1: "첫 목표를 시각화해보셨나요? 이렇게 하면 효과가 3배!" (사용 유도)
  - Day 2: "체험 D-1: 내일까지만 무료입니다" (긴급성)
  - Day 3: "체험이 종료되었습니다. 작성하신 목표 X개가 대기 중입니다" (손실 회피)

---

### 2-4. 4단계: 핵심 (Core) — Essential 전환

**목적:** 무료 체험에서 첫 유료 결제($9.99/월)로 전환

**전환 전략:**

| 전략 | 실행 내용 | 심리적 기제 |
|------|----------|------------|
| 가격 재프레이밍 | "하루 333원, 커피 반 잔 가격" | 앵커링 효과 |
| 손실 회피 | "체험 기간 세운 목표 X개가 삭제됩니다" | Loss Aversion |
| 사회적 증거 | "이미 X명이 Essential로 목표를 달성 중" | Social Proof |
| 한정 제안 | "첫 달 50% 할인" (최초 1회) | Scarcity |
| 무위험 보장 | "7일 내 환불 가능, 이유 불문" | Risk Reversal |

**전환율 목표:** 무료→Essential 15~25%

---

### 2-5. 5단계: 업그레이드 (Upsell) — Visionary 전환

**목적:** Essential 사용자를 Visionary($19.99/월)로 업그레이드

**업그레이드 트리거:**
- Essential 사용량이 한도의 70% 도달 시: "더 많은 시각화가 필요하신가요?"
- 영상 생성 시도 시 (Essential에 미포함): "Visionary에서만 가능한 AI 영상으로 꿈을 더 생생하게"
- 30일 이상 활성 사용자에게: "당신의 목표 달성률이 X%입니다. Visionary로 업그레이드하면 시각화 22배 + 영상 생성 해금"

**포지셔닝:**
- Essential = "AI 코치 시작" (코칭 + 기본 시각화)
- Visionary = "시각화 해금" (고품질 이미지 + 영상 + 대용량 음성)
- Master = 숨김 처리, VIP 제안으로만 노출 (희소성)

---

### 2-6. 6단계: 충성 (Loyalty) — 추천 & 입소문

**목적:** 만족한 사용자가 자발적으로 추천하는 바이럴 시스템

**전략:**
- **후기 시스템** (구현 완료): 목표 3개 달성 시 자동 프롬프트
- **추천 보상**: "친구 1명 추천 시 양쪽 모두 1개월 50% 할인" (향후 구현)
- **커뮤니티**: 목표 달성 성공 사례 공유 게시판 (향후 구현)
- **마일스톤 축하**: 10번째 목표 달성, 30일 연속 사용 등에 인앱 축하 + 소셜 공유 유도

## 5. 실행 로드맵

### Phase 1: 콘텐츠 기반 유입 (1~2주)
- [ ] 블로그 칼럼 5편 작성 및 발행
- [ ] 인스타그램/유튜브 숏폼 콘텐츠 5개 기획
- [ ] 무료 PDF 리드 자석 제작 (15페이지)

### Phase 2: 리드 수집 시스템 (2~3주)
- [ ] 이메일 수집 랜딩 페이지 제작 (또는 인앱 삽입)
- [ ] 7일 이메일 자동 시퀀스 설정
- [ ] 카카오톡 채널 개설 및 자동 메시지 설정

### Phase 3: 전환 최적화 (3~4주)
- [ ] 온보딩 플로우 개선 (첫 시각화 강제 체험)
- [ ] 체험 만료 이메일 시퀀스 구축
- [ ] 첫 달 할인 프로모션 설계

### Phase 4: 바이럴 시스템 (4~6주)
- [ ] 추천 보상 프로그램 구현
- [ ] 마일스톤 축하 시스템 구현
- [ ] 소셜 공유 기능 추가

## 6. 경쟁 우위 포지셔닝

### Secret Coach vs 경쟁 앱 차별점

| 기능 | 일반 투두앱 | 노션 | 코칭앱 | Secret Coach |
|------|-----------|------|--------|-------------|
| 마인드맵 목표 구조화 | X | 수동 | X | AI 자동 분해 |
| AI 1:1 코칭 | X | X | 일부 | 24시간, 맥락 기억 |
| 이미지 시각화 | X | X | X | AI 자동 생성 |
| 영상 시각화 | X | X | X | AI 8초 영상 |
| 음성 내러티브 | X | X | X | TTS 자동 생성 |
| 진행률 피드백 | 수동 | 수동 | 일부 | AI 자동 분석 |

**USP (Unique Selling Proposition):**
> "마인드맵 + AI 코칭 + 시각화(이미지/영상/음성)를 결합한 세계 유일의 목표 달성 플랫폼"

이 USP는 모든 콘텐츠와 광고에서 일관되게 사용할 것.

## 7. 이상한마케팅 프레임워크 체크리스트

- [x] **후크(Hook)**: 랜딩 헤드카피가 3초 안에 고통을 찌르는가?
- [x] **스토리(Story)**: 브랜드 뒤에 매력적인 캐릭터가 있는가? (향후 보강 필요)
- [x] **제안(Offer)**: 거절할 수 없는 제안 구조인가? (무료 체험 + 하루 333원)
- [x] **사회적 증거**: 수치와 후기가 제시되는가?
- [x] **반박 제거**: 핵심 의구심 5가지가 해소되는가?
- [x] **손실 회피**: "하지 않으면 잃는 것"이 명확한가?
- [ ] **긴급성**: 시간 제한 요소가 있는가? (프로모션으로 보강)
- [ ] **권위**: 전문가 인증/수상 경력이 있는가? (PR로 보강)
